Site web d’événements – PHP/MySQL, HTTPS

Vue d'Ensemble du Projet

Conception et développement d’un site web responsive permettant la recherche d’événements par ville, avec une architecture back-end PHP/MySQL. Mise en place des opérations CRUD, sécurisation des formulaires contre les injections, déploiement sur VPS via Docker, configuration d’Apache et activation du protocole HTTPS. Packshot d’un produit géorgien de Pâques

Packshot

Pour ce projet, j’ai réalisé un packshot d’un centre de table traditionnel de Pâques, combinant éléments culinaires et décoratifs L'objectif principal de ce projet était de vendre le produit et de le rendre intéressant pour les clients à l'acheter, ce projet m'a donc permis d'en apprendre davantage sur la stratégie marketing produit ainsi que sur l'aspect photographie. Ensuite, j’ai utilisé Adobe Lightroom pour Corriger l’exposition et la balance des blancsTravailler la lumière, et le contraste et les couleurs pour faire ressortir les textures Enfin, j’ai poursuivi le travail dans Adobe Photoshop pour Appliquer des retouches fines (nettoyage, détails)

Conception d’un gabarit UI pour un site de métropole (approche Mobile First)

J’ai conçu un gabarit pour un site web fictif d’une métropole, en privilégiant une approche Mobile First.

Gabarit

Dans le cadre de mon apprentissage en design d’interface, j’ai conçu un gabarit pour un site web fictif d’une métropole, en privilégiant une approche Mobile First. J’ai choisi de travailler sur la page des services, considérée comme essentielle pour les utilisateurs. Après un zoning papier, j’ai développé une grille de mise en page et une grille typographique en m’appuyant sur les échelles recommandées par Gridlover : tailles de texte entre 14 et 20 px, interlignage de 1,5 à 2x, marges latérales de 150 px sur desktop et 20 px sur mobile/tablette. Ce projet m’a permis de renforcer mes compétences sur Figma, de découvrir Gridlover, et d’acquérir une première expérience en UI design. J’ai ainsi compris l’importance de maquettes bien pensées pour faciliter le développement web.
